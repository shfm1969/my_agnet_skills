# Agent Skills 清單

以下整理了目前收集到的所有 Agent Skills 及其功能簡述。

| Skill 名稱 | 功能簡述 | 資料夾 |
| :--- | :--- | :--- |
| **nextjs-security-scan** | 針對 Next.js 專案進行全方位安全檢測 (OWASP 漏洞、Secrets、相依套件)。 | `01coder-agent-skills-main/skills/nextjs-security-scan` |
| **python-security-scan** | 針對 Python 專案 (Flask, Django, FastAPI) 進行漏洞掃描與注入檢測。 | `01coder-agent-skills-main/skills/python-security-scan` |
| **docx** | 讀取、解析與建立 Microsoft Word (.docx) 文件。 | `claudecode-skills-main/skills/docx` |
| **pdf** | 讀取與提取 PDF 文件中的文字與表單內容。 | `claudecode-skills-main/skills/pdf` |
| **pptx** | 讀取、解析與建立 PowerPoint (.pptx) 簡報。 | `claudecode-skills-main/skills/pptx` |
| **xlsx** | 讀取、解析與建立 Microsoft Excel (.xlsx) 試算表。 | `claudecode-skills-main/skills/xlsx` |
| **algorithmic-art** | 使用程式碼 (SVG/Canvas) 生成演算法藝術圖案。 | `claudecode-skills-main/skills/algorithmic-art` |
| **brand-guidelines** | 根據特定的品牌準則與語氣，協助檢查或撰寫內容。 | `claudecode-skills-main/skills/brand-guidelines` |
| **canvas-design** | 產生 HTML5 Canvas 圖形設計與繪圖程式碼。 | `claudecode-skills-main/skills/canvas-design` |
| **doc-coauthoring** | 協助多方協作的文件撰寫、修訂與整合。 | `claudecode-skills-main/skills/doc-coauthoring` |
| **frontend-design** | 協助設計現代化的前端 UI 元件與介面。 | `claudecode-skills-main/skills/frontend-design` |
| **internal-comms** | 撰寫專業的企業內部通訊、公告或電子郵件。 | `claudecode-skills-main/skills/internal-comms` |
| **mcp-builder** | 協助設計與建構 Model Context Protocol (MCP) 伺服器與工具。 | `claudecode-skills-main/skills/mcp-builder` |
| **skill-creator** | 引導並協助建立新的 Agent Skill 定義檔與結構。 | `claudecode-skills-main/skills/skill-creator` |
| **slack-gif-creator** | 根據文字描述產生適用於 Slack 的 GIF 動畫。 | `claudecode-skills-main/skills/slack-gif-creator` |
| **theme-factory** | 為應用程式或網站產生配色方案與設計主題系統。 | `claudecode-skills-main/skills/theme-factory` |
| **web-artifacts-builder** | 協助建構完整的網頁、元件或單頁應用成品。 | `claudecode-skills-main/skills/web-artifacts-builder` |
| **webapp-testing** | 針對 Web 應用程式進行自動化端對端測試 (如使用 Playwright)。 | `claudecode-skills-main/skills/webapp-testing` |
